# 使用说明
1. 下载项目
2. 按照提示安装游戏
3. 点击启动游戏
4. 按提示点击选择前往设置，设置LE路径
5. 再次启动游戏
## 这个项目的目的是给不会电脑的小白一个易于上手的图形化操作界面来启动，安装，修改游戏
